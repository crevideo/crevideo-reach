# 品牌知识包 — 模板（可选）

> 🇨🇳 **中文版（本页）** · 🇺🇸 [English](./00-brand-knowledge-pack-template.md)

> **可选**。填了 skill 会读，输出更贴品牌；不填 skill 走通用判断，照样能用。
> ⚠️ **这里只放品牌专属的东西 — 别放政策数字**。佣金上限、take-rate 闸门、车道门槛之类的数字放在「操作台 Operations Code → 关键规则速查（Critical-Rules Reference）」里，别在这重复一次，不然两边对不上。
> 用法：放进 Cowork Project 里（跟 skill 同一个 Project）。三段按需填，留空或 `(待填)` 都行。

---

## ① 品牌基本盘（fit-scoring / lane / 所有 skill 的底层依据）
- 品牌名 / 一句话定位：(待填)
- 主营类目：(比如 美妆 / 护肤)
- 目标受众：(年龄 / 性别 / 偏好 / 价位带)
- 当前主推商品（ACTIVATE 状态）：
  - SKU 1 — 名称 / 类目 / 售价 / 一句话卖点
  - SKU 2 — …
- 排除项 / 不合作的达人类型：(比如 跟某竞品深度绑定的达人 / 某类内容)

## ② 触达风格（outreach-message-composer / brand-safety 用）
- 品牌语气样本（2–3 行真实文案）：(待填)
- Do（鼓励的措辞）：(比如 真实使用感受、具体场景)
- Don't（除了全局红线之外，本品牌特有的避雷词 / 措辞）：(比如 不说 "medical-grade（医用级）")
- 偏爱的 CTA：(比如 "If you want to try it, just reply and I'll send over the partnership details")
- 本地化 / 语言：(默认美式英语；如果不一样写明)

## ③ 本季重点（fit / lane / outreach 用）
- 本月 / 本季优先级商品 + 排名：(待填)
- 本季排除（不推的 SKU / 类目）：(待填)
- 季节性钩子：(比如 节日、换季)
- 特别说明：(比如 某个 SKU 库存紧，限制触达量)

---
*每个 skill 只读它需要的那段（比如 reply-triage 一般不用这个 Pack）。这份 Pack 跟「关键规则速查」职责不重叠：这里说的是"品牌长啥样"，那边说的是"规则和数字"。*
