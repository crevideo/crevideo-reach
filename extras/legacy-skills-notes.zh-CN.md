# 专家模式（Skills）— 启用说明

> 🇨🇳 **中文版（本页）** · 🇺🇸 [English](./legacy-skills-notes.md)

> Claude 自带一组**专家模式**。你照常用手册第 2 部分的咒语，Claude 会在背后**自动启用**对应的专业流程 — **不用记名字，也不用手动挑。**

## 它们能帮你干啥
覆盖这些场景（咒语匹配时自动启用）：**达人打分**、判**车道**、**佣金 + 合规**审查、文案**合规扫描**、起草**触达文案**、**回复分诊**、**唤回 + 汰换**、**跑完任务后的效果诊断**。

## 怎么启用
- **最省事**：让 Crevideo 对接人帮你预装一次（或预打包在 Project 里）—— 之后你啥都不用动。
- **自己装**：Claude Desktop → **Settings → Skills** → 把这个目录的模块一个一个加上。
- **不装也能用**：15 分钟首胜和日常咒语**不依赖**专家模式 —— 启用只是让输出更专业、更稳、更一致。

## 可选：告诉 Claude 你的品牌（贴合度更高）
这个目录里的 `00-brand-knowledge-pack-template` 让你填品牌语气、避雷词、本季主推 SKU。填了产出更贴品牌；不填也照常能用。

*Crevideo 提供。*
